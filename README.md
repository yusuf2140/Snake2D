# Snake2D Engine

Snake2D is a lightweight, fast, and versatile 2D game engine built in Python. It is designed for rapid development, easy learning, and full control over your 2D games. Perfect for hobbyists, indie developers, and anyone who wants to bring their 2D worlds to life.

## Features

- Draw triangles and squares easily
- Track objects for deletion or updates
- Keyboard input support
- Simple and lightweight design
- Fully open-source and customizable
